Lottie Button Animations Demo

Arrow Animation Buttons

These buttons showcase the arrow.json Lottie animation that plays on hover and reverses on hover-off:

<button id="arrow-btn" class="btn-with-svg btn-with-arrow-animation">
    Download PDF
</button>

<script>
    document.addEventListener('DOMContentLoaded', function() {
        const arrowBtn = document.getElementById('arrow-btn');
        initArrowAnimation(arrowBtn);
        
        // The following behaviors are automatically included:
        // 1. Animation plays forward on hover
        // 2. Animation reverses on hover-off
    });
</script>

Scribble Animation Buttons

These buttons showcase the scribble.json Lottie animation that plays on hover. The animation reverses on hover-off, and the text changes to white color while hovering:

<button id="scribble-btn" class="btn-with-svg btn-with-scribble-animation">
    <span class="relative z-10">Contact Us</span>
</button>

<script>
    document.addEventListener('DOMContentLoaded', function() {
        const scribbleBtn = document.getElementById('scribble-btn');
        initScribbleAnimation(scribbleBtn);
        
        // The following behaviors are automatically included:
        // 1. Animation plays forward on hover
        // 2. Animation reverses on hover-off
        // 3. Text color changes to white on hover
        // 4. Text color returns to original on hover-off
    });
</script>
